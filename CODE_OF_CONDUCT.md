# Code of Conduct

This repository is intended for technical review, security analysis, and discussion of the Toqen.app mobile client.

## Expected behavior

Participants are expected to:

- communicate in a respectful and professional manner
- focus on technical and security-related topics
- provide constructive and evidence-based feedback
- avoid personal attacks, harassment, or disruptive behavior

## Scope

This code of conduct applies to:

- GitHub issues
- discussions
- any public interaction related to this repository

## Security-related communication

Security vulnerabilities must not be disclosed publicly.

Please follow the responsible disclosure process described in `SECURITY.md`.

## Enforcement

Maintainers reserve the right to:

- remove inappropriate content
- restrict participation if behavior violates these guidelines

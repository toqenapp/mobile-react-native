# Mobile FAQ

This FAQ covers the Toqen.app mobile client.

## Is The Mobile App Final Authorization Authority?

No. The mobile app presents request context, collects user intent, and signs approval challenges. Toqen.app server-side services verify requests and return final outcomes.

## Does Scanning A QR Grant Access?

No. QR parsing only lets the app resolve current request context.

## Where Are Platform Docs?

Platform-wide architecture, protocol, trust, privacy, and lifecycle docs are maintained outside this mobile repository.
